> A simple inventory application built with Express (Node.js)

### ✨ [Demo](https://disco-inventory-application.herokuapp.com/)

## Install

```sh
npm install
```

## Enviroment variables

```sh
touch .env
```

Check .env.example for example .env config

## Usage

```sh
npm run serverstart
```
